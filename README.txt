Репозиторий для распознавания текста на изображениях.
Данный репозиторий сделан на базе PaddleOCR.

В целом работа скрипта состоит из следующих этапов:

1) Найти области с текстом на изображении (бинарная классификация)
2) Получить рамки с областями текста
3) Вырезать области
4) Обработать вырезанные области моделями для распознавания текста

Для того чтобы протестировать работы нужно: 
1) создать venv
2) установить библиотеки из requirements.txt
3) запустить main.py


На текущий момент в репо сделано:
1) 2 моделели для поиска текста (detection) одна под англоязычные тексты и одна мультиязычная.
2) 2 модели для распознавания текста (recognition), одна для русского языка и одна английского.

Onnx модели получены путем экспорта моделей с помощью библиотеки paddle2onnx. Подробнее про экспорт моделей в соседнем репозитории (ocr_paddle_inference).

det_dyn_eng.onnx - модель для поиска текста ./models/det/en/en_PP-OCRv3_det_infer
det_dyn_mult.onnx - модель для поиска текста ./models/det/ml/Multilingual_PP-OCRv3_det_infer

rec_dyn_eng.onnx - модель для распознавания текста ./models/rec/en/en_PP-OCRv3_rec_infer
rec_dyn_rus.onnx - модель для распознавания текста ./models/rec/cyrillic/cyrillic_PP-OCRv3_rec_infer
